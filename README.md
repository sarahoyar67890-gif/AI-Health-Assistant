# 🧠 AI Health Assistant (ML Project)

An AI-powered web application that predicts possible diseases based on user-selected symptoms using Machine Learning.

Built using **Python, Scikit-learn, and Streamlit**.

## 📌 Project Overview

This project takes multiple symptoms as input and predicts the most probable disease using a machine learning model trained on a symptom-disease dataset.

It also shows **top 3 disease predictions with confidence scores**.

---

## 🧠 Tech Stack

- Python 🐍
- Pandas & NumPy
- Scikit-learn (Random Forest + TF-IDF)
- Streamlit (Web App)
- GitHub (Version Control)

---

## ⚙️ How It Works

1. User selects symptoms
2. Symptoms are converted into text format
3. TF-IDF converts text into numerical features
4. Random Forest model predicts disease
5. Top predictions are shown with confidence scores

---

## 📊 Model Used

- RandomForestClassifier
- TF-IDF Vectorizer
- Train/Test Split (80/20)

---

## 🖥️ Features

- Simple UI using Streamlit
- Multi-symptom selection
- Top 3 disease predictions
- Confidence scores
- Fast and responsive
